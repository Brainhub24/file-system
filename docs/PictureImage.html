PictureImage

PictureImage (context, data)

lowlight.fs.js (line 1238) 

Image data structure.
It allows to easily manipulate bytes on a image to store data on it.
This is intended to work together with a basic interface as a file system.

Unlike Image, this one can be used on top of an existing picture to achieve steganography.

  • Arguments

    context CanvasRenderingContext2D Canvas context
    data
    Default : null     		Object Format data (leave blank if you're loading an existing image)
